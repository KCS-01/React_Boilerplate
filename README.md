# React_BoilerPlate

## 설정 라이브러리

-   `zustand` : 전역 상태관리
-   `@tanstack/react-query` : 비동기 상태관리
-   `axios` : 요청
-   `react-router-dom` : 라우팅
-   `styled-components` : 스타일링
-   `styled-reset` : 스타일링 초기화
-   `dotenv` : 환경변수 사용

## 디렉토리 구조

```
./src
   ├─ /apis          ───> api요청 관리
   ├─ /assets        ───> Png, Svg, Font 등의 설정 관리
   ├─ /components    ───> 컴포넌트 관리
   ├─ /configs       ───> 앱 설정 관리
   ├─ /constants     ───> 앱 상수 관리
   ├─ /hooks         ───> 훅(로직) 관리
   ├─ /pages         ───> 주요 페이지 단위 컴포넌트 관리
   ├─ /routes        ───> 라우팅 관리
   ├─ /stores        ───> 전역 상태 관리
   └─ /styles        ───> 전역 스타일링 관리
```
